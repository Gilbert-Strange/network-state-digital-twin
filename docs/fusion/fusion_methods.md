# Fusion Methods  
  
## Classical Fusion  
  
### Feature-level Fusion  
- Combine features from multiple sources into one vector  
- Simple and widely used  
  
### Decision-level Fusion  
- Combine outputs of different models  
- More flexible but less direct 

### Hybrid-level Fusion
- Combine & Switch data
- Better but hard to design process
  
---  
  
## Deep Learning Fusion  
  
### Encoder-Decoder Fusion  
- Features are fused in latent space  
  
### Attention-based Fusion  
- Assign different importance to different features  
- Dynamic weighting  
  
### Graph-based Fusion  
- Use graph structure to model relationships

### GenNN-based Fusion
- For generate data we loss
- Eliminate the overfitting used as a predictive model

### Other Constraint-based Fusion
- Set constraint
